# Projeto: Landing Page Interativa

## 🚀 Sobre o projeto

Este projeto é uma landing page desenvolvida com HTML, CSS e JavaScript, com foco em praticar manipulação do DOM, responsividade e criação de interações com o usuário.

O objetivo foi simular um site real com elementos interativos e layout moderno, aplicando boas práticas de desenvolvimento front-end.

## ⚙️ Tecnologias

- HTML5
- CSS3
- JavaScript

## ✨ Funcionalidades

- Layout responsivo
- Interação com botões
- Mensagens dinâmicas na tela
- Design moderno e limpo

## 🎯 Aprendizados

- Manipulação do DOM
- Estruturação de layout
- Organização de código
- Lógica de interação com usuário
